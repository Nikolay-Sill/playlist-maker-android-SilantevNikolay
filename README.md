# playlist-maker-android-SilantevNikolay
Учебный проект по созданию мобильного приложения

## Версии

### v3.0 - Спринт 3
Создан главный экран MainActivity по дизайн-макету.
Созданы SettingsActivity и SearchActivity, а также добавлен код для перехода на эти экраны.

### v4.0 - Спринт 4
Реализована вёрстка экранов поиска и настройки.
Реализован функционал кнопок экрана настроек.
Вся логика навигации перенесена в класс PlaylistHost.

### v5.0 - Спринт 5
Создали эмулятор сервера, слой Data и слой Domain, а также реализовали репозиторий.

### v6.0 - Спринт 6
Создан ViewModel.
Разработан полноценный экран поиска треков, построенный на MVVM и Jetpack Compose.

### v7.0 - Спринт 7
Создан экран, на котором размещена кнопка FloatingActionButton.
Создана функция, отображающая ModalBottomSheet.
При нажатии на FloatingActionButton появляется окно ModalBottomSheet.

### v8.0 - Спринт 8
Реализована история поиска.
Создана заглушка для базы данных.
Реализован переход на экран TrackDetails.

### v9.0 - Спринт 9
Реализовано выполнение настоящих поисковых запросов.
Реализован экран плейлиста.
