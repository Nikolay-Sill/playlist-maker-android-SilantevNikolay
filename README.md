# Playlist Maker Android

Мобильное приложение для создания и управления музыкальными плейлистами с возможностью поиска треков через iTunes API, сохранения избранного, истории поиска и работы с локальной базой данных.

## Технологии
- **Язык:** Kotlin
- **Минимальная версия Android:** 29 (Android 10)
- **Целевая версия Android:** 36
- **Архитектура:** Clean Architecture, MVVM
- **UI:** Jetpack Compose
- **Локальное хранение:** Room, DataStore
- **Сеть:** Retrofit, iTunes API
- **DI:** Koin
- **Асинхронность:** Kotlin Coroutines

## Сборка и запуск

### Требования
- Android Studio 2023.1+
- JDK 11+
- Android SDK 29+

### Шаги
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/Nikolay-Sill/playlist-maker-android-SilantevNikolay.git
2. Откройте проект в Android Studio.
3. Дождитесь завершения синхронизации Gradle и загрузки зависимостей.
4. Создайте виртуальное устройство (AVD) с Android 10+.
5. Запустите приложение на эмуляторе или физическом устройстве.
